# VHDL Mini LLM Core

A hardware-based text classification prototype implemented in VHDL. 

## Features

- Fixed-point arithmetic using Q4.4 and Q8.8 formats
- MAC-based hidden and output layer computation
- ReLU activation and argmax-based prediction
- FSM-controlled training and validation flow
- Weight and bias update logic
- ROM/register-based storage for training and validation samples
- Testbench simulation with waveform verification
